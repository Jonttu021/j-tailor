# j-tailor

- An immersive tailor job with custom minigame for FiveM servers.
- Players can collect materials, craft clothes, and earn money as a professional tailor.

- Built fully as a standalone resource with support for OX ecosystem tools.

#✨ Features
- 🧵 Collect materials
- 👕 Craft clothing items
- 🌍 Fully standalone
- 🛠 Custom Minigame

#This resource requires:

- ox_lib
- ox_target
- ox_inventory

📸 Preview
**https://youtu.be/p56WC8M089g**

Installation
1. Download the script
2. Place it in your resources folder
3. Ensure dependencies are installed
4. Add this to your server.cfg:

- ensure ox_lib
- ensure ox_target
- ensure ox_inventory
- ensure j-tailor
